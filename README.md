### 
Employee Management System
This project is an Employee Management System built with .NET and configured for database management to help streamline staff record tracking and department administration.

Features
Employee Management: Perform CRUD (Create, Read, Update, Delete) operations on staff records.

Department Tracking: Organize and filter employees based on their respective departments.

Data Integrity: Managed database schema using Entity Framework migrations.

Technical Stack
Framework: .NET

Frontend: Angular (used in the Catering Orders management application)

Task Management: Implements service interfaces and API controllers for task-tracking logic.

Getting Started
To get a local copy up and running, follow these steps:

Clone the repository:

Bash
git clone https://github.com/codewithfatima/Employee_Management_System.git
Restore Dependencies:
Open the solution in Visual Studio or run dotnet restore in your terminal.

Database Setup:
Run the following command to apply the migrations and set up your database:

Bash
dotnet ef database update
Run the Application:
Press F5 in Visual Studio or run dotnet run in the terminal.
